# Zakini Vintage

## How do I install these formulae?

`brew install zakini/vintage/<formula>`

Or `brew tap zakini/vintage` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "zakini/vintage"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
